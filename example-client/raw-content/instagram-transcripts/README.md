# Reel Transcripts

This folder holds Whisper transcripts of the creator's past Instagram reels. The skill loads every `.txt` file in here as the voice match corpus in Phase 2.

Transcripts are omitted from this public repo. In a real client setup this folder contains 10 to 20 `.txt` files, one per past reel, generated by `scripts/scrape-reels.py` plus a yt-dlp + Whisper pass.
